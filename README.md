# Geek_SougouIME
搜狗拼音输入法隐私保护增强版  
本人基于搜狗拼音输入法第三方修改版再度暴力修改  
**（原始版本10.2.0.4519）**  
目标着重于隐私保护  
本版本在第三方修改版的基础上，增加以下修改特性：
* 暴力阻止绝大部分尝试联网的请求，一刀切
* 禁止后台输入字数统计
* 调整右键菜单布局，删除没用的按钮和选项
* 调整输入框界面，删除使用手写输入提示
* 禁止记录、生成部分日志文件
* 禁止联网推荐皮肤
* 保留离线词库、皮肤安装功能
* 保留第三方账户登录功能（仅登录，不会上报数据）
* 所有修改不加壳不加花，保证安全透明

本版本适用于：  
 - **极度隐私敏感人士**
 - **防火墙联网审计爱好者**
 - **任务管理器查看员**
 - **电脑系统洁癖患者**
 - **~~国产软件~~（伪）爱好者**
 
使用方式  
1.先安装原始第三方修改版  
2.将三个可执行程序分别替换到对应的目录（可能需要WinPE、重新启动）  
3.enjoy~
